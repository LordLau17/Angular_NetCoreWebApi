# Angular-NetWebApi
# Required
- .NET 7.0
- Typescript 4.8
- Node v18

# Initial setup
- spa -> 'npm install'

# Generate Angular httpclient
- Run 'npm install -g nswag'
- Run any Nswag definition with: 'nswag run'
- or 
- Add new Nswag definition with: 'nswag new'

# Good to know
- API and SPA run on https;
